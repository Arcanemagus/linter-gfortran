# Linter-gfortran

A linter plugin for [Linter](https://github.com/AtomLinter/Linter), using
`gfortran` to lint fortran code.

This is based on [linter-gcc](https://github.com/hebaishi/linter-gcc/), and work
with gfortran 5.
